# 旋转教练 (RotationCoach)

实时分析你**实际按下的技能序列**，指出发呆、卡手、GCD 节奏问题，帮助你打磨手法。

- 钩取 `ActionEffectHandler.Receive`，只记录本地玩家实际按下的技能（不依赖伤害数字）。
- 自动按战斗分段，支持实时 / 历史双模式。
- 统计 APM、GCD 节奏（中位间隔）、发呆段数与最长停顿。
- 列出「待改进」：发呆段、疑似卡手（GCD 间隔远低于常态）。
- 完整出手序列 + 技能中文名。

> GCD 判定为启发式：动画锁 ≥ 1.2s 视为 GCD。发呆 / APM 等指标不依赖该判定，始终可靠。

## 命令
- `/rc` —— 打开旋转分析窗口
- `/rc reset` —— 清空历史出手记录
